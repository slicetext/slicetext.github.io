First Things to do in Lubuntu

June 24, 2023

1: Install Nautilus

I personally hate the default File Manager, PCMANFM-Qt. Because of this, I like to install gnome's default File Manager, Nautilus. This is super easy. Just run sudo apt install nautilus in the terminal.

2: Set your Theme

Lubuntu has many different themes. You can find them and choose your favorite in the LXQt menu by going to Preferences>LXQt Settings>Appearence. You can change the theme in the LXQt Theme section of the menu. I like to use the Lubuntu Arc theme.

3: Set your Icons to Numix Circle

I think this icons look very nice. You can use them by opening up the terminal and running sudo apt install numix-icon-theme-circle. Then, to apply them, open up the same Appearence menu as number 2 and instead navigate to the icon section and select your icons to be Numix Circle.

4: Change your Panel's Widgets

To do this, right click on your panel and select Manage Widgets. In this menu, you can press plus to add a widget, minus to remove one once you have clicked on it, and the arrows to move the widget you have clicked on.
